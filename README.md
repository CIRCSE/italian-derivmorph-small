# italian-derivmorph-small
The turtle files for italian derivational morphology. 

## The CIRCSE fork

This dataset is forked from the [original project](https://github.com/adamfengxx/italian-derivmorph-small) of X. Feng. All works for future publication of the dataset in LiITA will be done on this fork.

The original data are taken from the [derIvaTario](https://github.com/rahonalab/derIvaTario/tree/main).

Some statistics and an overview of the data can be found in the [explore.ipynb](https://github.com/CIRCSE/italian-derivmorph-small/blob/main/scripts/explore.ipynb) notebook.
